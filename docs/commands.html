All Commands

Complete reference for all BobbyTheBot Mafia commands. Use these to start games, view roles, and control gameplay.

üéÆ Primary Commands

!createmafia

Everyone

Start a new Bee Mafia game with all players currently in the voice channel.

Requirements:

  • Must be in the designated Mafia voice channel
  • Minimum 6 players required
  • No player can already be in an active game

Parameters:

  • random - Enable random role mode
  • revealroles - Show all roles to all players at game start
  • [preset_name] - Use a specific preset role distribution

Usage Examples:

!createmafia - Standard game with default roles
!createmafia random - Chaos mode with random roles
!createmafia revealroles - Everyone sees all roles from start
!createmafia basic - Use 'basic' preset
!createmafia chaos random - Chaos preset with random roles
After Running: An interactive button prompt appears to configure phase timings:
  • Quick Start - Uses default timings (Night: 60s, Day: 180s, Voting: 120s)
  • Configure Custom Times - Opens modal to set custom phase durations
  • Auto-Start - If no button clicked within 2 minutes, game starts with defaults

!mafiaroles or !roles

Everyone

View detailed information about all available roles in the game.

Parameters:

  • bee - Show only Bee team roles (town)
  • wasp - Show only Wasp team roles (mafia)
  • neutral - Show only Neutral roles
  • all - Show all roles (default if no parameter)

Usage Examples:

!mafiaroles - Shows all roles
!mafiaroles bee - Shows only Bee roles
!roles wasp - Shows only Wasp roles
!mafiaroles neutral - Shows only Neutral roles

Output Includes:

  • Role name and emoji
  • Role description
  • Key abilities (top 3)
  • Whether role has night action (üåô) or defense (üõ°Ô∏è)
  • Win conditions

!mafiapresets or !presets

Everyone

Display all available preset role distributions with descriptions.

Usage:

!mafiapresets - Shows all available presets
!presets - Alternative command

Common Presets:

  • basic - Classic balanced game with standard roles
  • chaos - Chaotic game with more neutrals and special roles
  • And more! Use the command to see all options

Use presets with: !createmafia [preset_name]

!reveal

Queen Bee Only

Reveal yourself as the Queen Bee to gain 3 extra votes (4 total) during voting phase.

Requirements:

  • Must be the Queen Bee role
  • Must be alive
  • Can only be used during day or voting phase
  • Only usable once per game (permanent)

Usage:

!reveal - Permanently reveals Queen Bee to all players
Effect: Your vote becomes worth 4 votes instead of 1 for the rest of the game. All players will know you are the Queen Bee.

üêõ Debug & Testing Commands

These commands are for testing and development purposes. They allow testing with bots and custom role configurations.

!createmafiadebug

Debug Mode

Create a test game with specified roles plus 12 bot players for testing and debugging.

Requirements:

  • Must be in the Mafia voice channel
  • Can have fewer than 6 real players (bots fill the rest)

Parameters:

  • [ROLE] - Assign a specific role to yourself
  • ROLE1,ROLE2 - Comma-separated for multiple players
  • random - Enable random mode for unspecified roles
  • NoAI - Disable bot auto-actions
  • revealroles - Show all roles at game start

Usage Examples:

!createmafiadebug SCOUT_BEE - Play as Scout Bee
!createmafiadebug SCOUT_BEE,WASP_QUEEN - Assign multiple roles
!createmafiadebug WASP_QUEEN random - Play as Wasp Queen with random mode
!createmafiadebug PIRATE_BEETLE NoAI - Bots don't auto-act
Valid Role Names: Use exact role names like SCOUT_BEE, WASP_QUEEN, MURDER_HORNET, etc. See !mafiaroles for full list.

!mafiadebugskip

Organizer Only

Immediately skip to the next game phase (debug mode only).

Requirements:

  • Game must be in debug mode
  • User must be the game organizer

Usage:

!mafiadebugskip - Skips to next phase immediately

Phase Progression: Setup ‚Üí Night ‚Üí Dusk ‚Üí Day ‚Üí Voting ‚Üí (repeat) or End

!mafiadebugend

Organizer Only

Immediately end the debug game and clean up all game state.

Requirements:

  • Game must be in debug mode
  • User must be the game organizer

Usage:

!mafiadebugend - Ends the current debug game
Note: This will clean up all game state, unmute players, and reset the channel.

üí¨ In-Game Interactions

During gameplay, players interact with the bot through DMs and buttons rather than commands.

üåô Night Actions

Players with night abilities receive DM prompts from the bot with numbered buttons to select targets.

  • Format: Bot sends a message with buttons for each alive player
  • Selection: Click the button corresponding to your target
  • Confirmation: Bot confirms your action in DM
  • Wasp Team: Vote on kill target in team chat

üó≥Ô∏è Voting

During voting phase, players vote through DM buttons sent by the bot.

  • Format: Bot sends buttons with all alive player names + "Skip" option
  • Selection: Click the player you want to vote to eliminate
  • Change Vote: Click a different button to change your vote
  • Results: Announced in game channel when voting ends

üí¨ Text Communication

Messages sent in the mafia text channel during day phase are relayed to all players.

  • Day Phase: All alive players can see text messages
  • Night Phase: Only Wasps can communicate in team chat
  • Dead Chat: Dead players can communicate with Medium Bee at night
  • Special Effects: Blackmailed/Deceived players have messages transformed

üé≠ Special Role Actions

Some roles have unique interaction methods:

  • Jailer Bee: Selects jail target at dusk, decides execution at night
  • Transporter Bee: Selects two players to swap at dusk
  • Pirate Beetle: Selects duel target and RPS choice at dusk
  • Clown Beetle: After being lynched, selects one voter to haunt
  • Queen Bee: Use !reveal command during day/voting

üí° Command Tips

Starting Your First Game

  • Use !createmafia basic for a balanced beginner game
  • Get 6+ players in voice channel first
  • Use "Quick Start" for default timings on your first game
  • Run !mafiaroles bee before starting to learn Bee roles

Testing & Learning

  • Use !createmafiadebug SCOUT_BEE to practice a specific role
  • Add revealroles to see what bots are doing
  • Use NoAI flag to manually control bot actions for testing
  • Use !mafiadebugskip to speed through phases while testing

During Gameplay

  • Always check your DMs for night action prompts
  • Submit actions quickly - timers are active!
  • Queen Bee: Save !reveal for crucial moments
  • Dead players: You can still help your team via Medium Bee
