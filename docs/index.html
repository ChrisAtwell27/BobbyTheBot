Quick Start Guide

1

Join the Game

Join a voice channel and wait for the game to start. You'll receive a DM with your role assignment.

2

Understand Your Role

Read your role description carefully. Check if you have night actions or special abilities.

3

Night Phase

If you have a night action, you'll receive a DM prompting you to select targets using buttons.

4

Day Phase

Discuss with other players, share information, and try to identify evil players.

5

Voting Phase

Vote to eliminate a player using the buttons in the game channel. Majority wins!

ğŸŒ™ Night Actions

Many roles can perform actions at night:

  • Killing: Wasps vote on who to kill, Serial Killers choose targets
  • Investigating: Learn information about other players
  • Protecting: Heal or guard players from attacks
  • Roleblocking: Prevent a player's action from working
  • Visiting: Most actions involve "visiting" a target

ğŸ—³ï¸ Voting & Elimination

  • Each player gets 1 vote during voting phase
  • Queen Bee gets 4 votes after revealing
  • Minimum 2 votes needed to eliminate (1 in debug mode)
  • Ties: Trigger a 60-second revote, then random selection if still tied
  • Marshal Bee can protect one Bee per day from lynching
